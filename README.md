# Life Expectancy and GDP Analysis

This project analyzes the relationship between GDP and life expectancy across six countries (Chile, China, Germany, Mexico, USA, and Zimbabwe) from 2000 to 2015 using Python and pandas.

## Files

- `all_data.csv` — dataset  
- `life_expectancy_analysis.ipynb` — analysis notebook  
- Visualization outputs: `avg_gdp_life_exp.png`, `life_expectancy.png`, `gdp.png`, `gdp_vs_life_expectancy.png`

## Key Insights

- Life expectancy generally rises over time; Zimbabwe remains lower overall.  
- GDP growth is strongest in China and the USA; Zimbabwe has the lowest GDP.  
- Higher GDP tends to correlate with higher life expectancy across countries.
